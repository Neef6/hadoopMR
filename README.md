2.物品协同过滤的实现
    推荐结果=物品的同现矩阵A * 用户对物品的评分矩阵B
    B：如果两个物品经常同时被很多用户喜欢，那么可以说这两个物品是相似的，同时被越多的用户喜欢（即为通同现度，上面矩阵中的值），这两个物品的相似度就越高。 
    
    同好矩阵（A）
                                
            101 102 103
    
    101
    
    102
    
    103
    
    用户评分矩阵（B）
    
            1 2 3
    
    101
    
    102
    
    103
